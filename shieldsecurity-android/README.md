# ShieldSecurity Android

Новый Android-проект на Kotlin + Jetpack Compose Material 3.

Что уже есть:

- вход по почте и паролю
- регистрация: имя, почта, пароль, повтор пароля
- кнопки `Нет аккаунта?`, `Забыли пароль?`, `Есть аккаунт?`
- гостевой режим
- отдельный экран ввода 6-значного кода
- глазик у полей пароля
- главная страница с блоком угроз
- режимы `Быстрая`, `Глубокая`, `Выборочная`
- `Глубокая` доступна только после входа
- блок `Проверить APK`
- верхняя панель с `История` и заголовком `ShieldSecurity`
- экран истории проверок
- плавные переходы и загрузочный экран

Важно:

- сейчас это стабильный UI-прототип и локальная логика приложения
- для подтверждения входа в прототипе используется код `111111`
- реальная отправка кода на email и реальный антивирусный движок для Android ещё не подключены
- APK-проверка и скан-режимы сейчас работают как демонстрационный поток интерфейса

Открывать в Android Studio:

1. Открыть папку `shieldsecurity-android`
2. Дождаться синхронизации Gradle
3. Запустить модуль `app`

Если Android Studio попросит Gradle Wrapper, можно разрешить ей создать/обновить wrapper автоматически.
